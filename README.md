# HTML Basics – Links, Lists and Images

## Project Overview

This repository contains simple HTML files created as part of Full Stack Development (FSD) practice.

The project demonstrates the basic usage of:

- Hyperlinks
- Lists
- Images
- Thumbnail images

All files are created using pure HTML to understand fundamental web development concepts.


## Objectives

- Learn how to create hyperlinks in HTML
- Understand different types of lists
- Display images in a webpage
- Create thumbnail images
- Practice basic HTML structure


## Technologies Used

- HTML5


## Project Structure

```
HTML-Links-Lists-Images/
├── hyperlink.html
├── list.html
├── image.html
├── thumbnail.html
└── README.md
```


## File Description

### 1. hyperlink.html
Demonstrates how to create hyperlinks using the anchor tag.
Includes examples of:
- External links
- Internal links
- Opening links in a new tab

### 2. list.html
Demonstrates different types of lists:
- Ordered list
- Unordered list
- Description list

### 3. image.html
Shows how to insert images into a webpage using the img tag.
Includes:
- Image source
- Alt text
- Width and height attributes

### 4. thumbnail.html
Demonstrates how to create thumbnail images.
Includes:
- Smaller preview image
- Clickable image linked to full-size version


## How to Run

1. Download or clone the repository:

```
git clone <repository-link>
cd HTML-Links-Lists-Images
```

2. Open any HTML file in a web browser.

No additional software or dependencies are required.
